# Decathlon_clone
A clone of an e-commerce website where you can buy sports related things like sportswear,sportsshoes,gym related items etc.
​
### Tech stacks used :

*  **Html** 
*  **CSS** 
*  **Advanced Java Script** 
*  **Express JS**
*  **Mongo DB**
